Домашнее задание 1: MyHashMap

Реализованные методы:
- `put(K key, V value)` - добавление/обновление пары
- `get(K key)` - получение значения
- `remove(K key)` - удаление пары

Особенности:
- Обработка коллизий через цепочки (chaining)
- Пример коллизии: ключи "Aa" и "BB" имеют одинаковый хэш
- Начальный размер таблицы: 16
